# PATHFINDER    

A pathfinder algorithm using A* search algorithm

![Image of Yaktocat](https://media.giphy.com/media/MBXxK8vMKk7HossyYn/giphy.gif)

# CONTROLS

button | function
-------- | ---------
S | set a starting point
G | set a goal
W | set an obstacle
T | remove obstacle
C | clear all
ENTER | shows realtime searching
